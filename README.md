# kidsCoding

A responsive kids coding learning platform repository.

Current repository contents:
- MVP product spec: `docs/superpowers/specs/2026-03-30-kids-coding-mvp-design.md`
- MVP implementation plan: `docs/superpowers/plans/2026-03-30-kids-coding-mvp.md`
- MVP web app: `apps/web`

Current MVP scope:
- responsive Web experience for mobile and desktop
- onboarding with age routing and lightweight assessment
- guided Blockly lesson flow and first animated story project
- collectible cards and progress tracking
- guest progress save and account binding
- parent overview page
